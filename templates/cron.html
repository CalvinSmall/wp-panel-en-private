Replaces WordPress Built-in Pseudo Cron — By default WordPress triggers scheduled tasks (publish scheduled posts, check for updates, etc.) only on page visits, which is unreliable and resource-intensive. This task invokes wp-cron.php via system cron on a fixed schedule for better reliability.

                    

                        
                        
                    

                    
On creation, define('DISABLE_WP_CRON', true) will be added to wp-config.php to disable WordPress built-in pseudo cron. This config will be automatically removed when the task is deleted (and no other WP Cron tasks exist for this site).
