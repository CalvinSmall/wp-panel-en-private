# 更新说明

## v1.2.9

### 新增
- 面板更新增加实时进度提示，更新过程中显示下载进度、校验状态和安装阶段

### 优化
- 文件管理器压缩、解压大文件时显示处理中提示，并在任务完成前禁用重复操作按钮
- 面板更新下载校验文件或签名文件失败时，提示用户可能是 GitHub 或服务器网络临时异常，可稍后重试
- 网站详情页修改站点 URL 的注意事项补充浏览器缓存提示，避免用户被旧站跳转误导
- 网站详情页数据库卡片修改站点 URL 后自动刷新 Nginx FastCGI 缓存并清理 Redis Object Cache，减少旧域名跳转残留

### 修复
- 面板更新的 SHA256 校验文件和签名文件下载增加轻量重试，减少 GitHub Release 下载链路短暂抖动导致的更新失败
- 修复网站详情页编辑域名时日志目录重命名失败会导致保存失败的问题；无法迁移旧日志目录时会自动创建新域名日志目录和日志文件
