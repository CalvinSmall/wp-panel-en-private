## WP Panel v1.2.0

六一来临，送给所有 WordPress 站长的一份礼物。

我从 2009 年开始用 VPS 搭 WordPress，军哥 lnmp、宝塔一路用过来。我的博客专门教新手建站，很多读者不是计算机专业，服务器管理对他们来说是道坎。

这两年 AI 编程让我看到了机会。一个不懂 Go 语言的人，居然真把面板写出来了。WP Panel 只做一件事：帮你在 VPS 上管好 WordPress。不做臃肿功能，新手也能上手。

多年靠 WordPress 生态吃饭，这是我回馈圈子的一点心意。永久免费，代码开源。

项目是 AI 写的、我测的，三轮安全审计修了 44 个问题。它还不够成熟，但我会持续改进。如果你愿意试一试，遇到任何问题随时反馈，你在帮这个项目变得更好。

—— naibabiji

### v1.2.0 新增

**部署**
- 新增 `install-cn.sh` 国内优化版安装脚本：使用上海交大镜像 + 自建反代，解决国内服务器安装时 APT 源和 GitHub 无法访问的问题
- 面板后台新增 GitHub 反代地址设置：支持自定义反代 URL，含测试按钮，国内用户版本检查/下载/校验全链路通过反代请求
- 安装脚本 `apt-get` 显示完整进度，便于排查问题
- 安装完成后同时显示公网/内网 IP
- MariaDB 密码优先读取已有值，防止安装中断重装时密码不一致

**安全**
- 新增 SSH 22 端口扫描防护：每 IP 超过 3 次/分钟新连接自动内核丢弃
- nftables 初始化代码去重，统一收归 `executor.EnsurePersistNftables()`

**配套升级**
- README 同步代码实际情况，移除 Debian 12 支持声明
- 需求文档同步全部功能变更
- 新增 Cloudflare Worker 自建反代部署指南

**重要说明**
- 正式版仅支持 Debian 13 (Trixie)，不再兼容 Debian 12
- 国内用户安装后建议在面板设置中填写反代地址，否则无法检测更新
